#LazyAdmin

> LazyAdmin是基于SSM的后端管理系统中权限管理功能的脚手架

## 技术选型
* Spring 4.2.6
* Mybatis Plus 2.0
* Shiro 1.2.2
* Hibernate Validator 5.1.3
* Druid 1.0.19
* Redis 2.3.1
* Ehcache 2.6.1
* Freemarker 2.3.20
* JDK 1.7
* Tomcat 8
* Maven
* Bootstrap 3


## 项目功能

* 基于Shiro的用户登录校验及session定时检查
* 细粒度的权限控制，精确到按钮权限
* 实现常规实体内容的增删改查
* 权限异常统一处理
